=== WooCommerce Digital Products ===
Contributors: duttaprabhaker-git
Tags: woocommerce, digital products, license keys
Requires at least: 5.0
Tested up to: 6.6
Stable tag: 1.0.0

Adds license key generation and secure downloads for WooCommerce digital products.

== Description ==
- Auto-generate license keys per order
- Secure, expiring download URLs
- Custom order meta data

== Installation ==
1. Upload the plugin to /wp-content/plugins/
2. Activate through WordPress Plugins menu
